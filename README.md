# node-starter

Node Simple API Application

- express
- express routes
- nodemon
- mongoose
- require-dir
- mongodb crud
- mongoose paginate
- cors

## Dependencies

```
$ sudo docker pull mongo
$ sudo docker run --name mongodb -p 27017:27017 -d mongo
```

## Tips

- Use Insomnia to test the application.
- Use Robot3T to access the MongoDB database.
